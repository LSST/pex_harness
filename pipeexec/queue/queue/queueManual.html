lsst.apps.mw.dps.pipeexec.queue Manual

Introduction

The lsst.apps.mw.dps.pipeexec.queue routines are used in conjunction with each other to move files into a directory and retrieve files in the order they were place there. There are two methods:

  • put - moves a file into a new directory

    • This method takes two arguments: the name of the file you want to move, and the directory where the file will be moved.

  • get - gets the oldest file in this directory. If there are no files in this directory, block until a file appears.

    • This method takes one argument: the directory you want to watch. It returns with a filename.

Example Usage

Two example programs in pymw/examples/queue/, queueget.py and queueput.py, show how methods in queue.py work. The following is an example of how they're used.

  1. Open two shell windows.

  2. In the first window, create two directories:

    3. $ mkdir tempdir tempdir2

  3. In the first window, execute the queueget.py program:

    4. $ python queueget.py tempdir2

  4. In the second window, put a file into the tempdir directory

    5. $ echo >tempdir/file1

  5. In the second window, execute the queueput.py program:

    6. $ python queueput.py tempdir/file1 tempdir2

You should see the message: 
        got tempdir2/file1
in the first window.
